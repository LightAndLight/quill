Quill takes the guesswork out of SQL.

Quill is a language that gives you confidence in your queries.

Quill makes it easy to write composable, correct queries, no matter your SQL database or backend programming language.

Get Started

type DOB = { year : Int, month : Int, day : Int };

table Person {
  id : Int, PrimaryKey(id), AutoIncrement(id),
  name : Text,
  dob : DOB
}

select_person_by_id(id : Int) -> Query (Rows Person) {
  persons <- select from Person;
  return (
    for person in persons
    yield person
    where person.id == id
  );
}
